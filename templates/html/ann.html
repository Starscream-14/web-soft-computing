Jaringan Saraf Tiruan (ANN)

Pengertian

Artificial Neural Network (ANN) atau Jaringan Saraf Tiruan adalah model komputasi yang terinspirasi dari cara kerja neuron biologis dalam otak manusia. ANN terdiri dari unit-unit pemrosesan yang saling terhubung (neuron tiruan) yang dapat belajar untuk mengenali pola kompleks dalam data.

ANN bekerja dengan cara menerima input, memprosesnya melalui lapisan-lapisan neuron dengan bobot tertentu, dan menghasilkan output. Melalui proses training, ANN dapat menyesuaikan bobot-bobotnya untuk meningkatkan akurasi prediksi.

Komponen Utama ANN

  • Input Layer: Lapisan yang menerima data input awal
  • Hidden Layer(s): Lapisan tersembunyi yang memproses informasi dengan bobot dan fungsi aktivasi
  • Output Layer: Lapisan yang menghasilkan prediksi akhir
  • Weights (Bobot): Parameter yang menentukan kekuatan koneksi antar neuron
  • Activation Function: Fungsi yang menentukan output neuron (sigmoid, ReLU, tanh, dll)
  • Bias: Nilai offset untuk menyesuaikan threshold aktivasi

Aplikasi ANN

  • Pengenalan gambar dan wajah
  • Klasifikasi teks dan sentiment analysis
  • Prediksi harga saham dan time series
  • Sistem rekomendasi
  • Speech recognition dan natural language processing
  • Medical diagnosis

Implementasi Python

Berikut adalah implementasi sederhana ANN dengan 2 input, 3 hidden neurons, dan 1 output menggunakan NumPy:

backend/server.py - Class ANN Sederhana 
import numpy as np

class SimpleANN:
    """Jaringan saraf sederhana: 2 input → 3 hidden → 1 output"""
    
    def __init__(self):
        # Bobot pre-trained untuk demo
        # Dalam aplikasi nyata, bobot ini hasil dari training
        self.w_input_hidden = np.array([
            [0.5, -0.3, 0.8],   # Bobot dari input 1 ke 3 hidden neurons
            [0.2, 0.6, -0.4]    # Bobot dari input 2 ke 3 hidden neurons
        ])
        self.b_hidden = np.array([0.1, -0.2, 0.3])  # Bias hidden layer
        
        self.w_hidden_output = np.array([[0.7], [-0.5], [0.9]])  # Bobot ke output
        self.b_output = np.array([0.2])  # Bias output layer
    
    def sigmoid(self, x):
        """Fungsi aktivasi sigmoid: σ(x) = 1 / (1 + e^(-x))"""
        # Clip untuk mencegah overflow
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))
    
    def forward(self, inputs):
        """Forward propagation: hitung output dari input"""
        
        # 1. Hidden layer computation
        # h = σ(W_input × inputs + bias_hidden)
        hidden_input = np.dot(inputs, self.w_input_hidden) + self.b_hidden
        hidden_output = self.sigmoid(hidden_input)
        
        # 2. Output layer computation
        # output = σ(W_hidden × hidden + bias_output)
        final_input = np.dot(hidden_output, self.w_hidden_output) + self.b_output
        final_output = self.sigmoid(final_input)
        
        return final_output[0], hidden_output


# Inisialisasi model global
ann_model = SimpleANN()


@app.route('/api/ann', methods=['POST'])
def ann_demo():
    """Endpoint API untuk demo ANN"""
    try:
        data = request.get_json()
        x1 = float(data.get('x1', 0.5))
        x2 = float(data.get('x2', 0.3))
        
        # Buat array input
        inputs = np.array([x1, x2])
        
        # Forward propagation
        output, hidden = ann_model.forward(inputs)
        
        return jsonify({
            "inputs": [x1, x2],
            "hidden_layer": hidden.tolist(),
            "output": round(float(output), 4),
            "explanation": f"Input [{x1}, {x2}] diproses melalui 3 neuron tersembunyi. "
                          f"Aktivasi hidden layer: {[round(h, 3) for h in hidden]}. "
                          f"Output akhir: {round(float(output), 4)}."
        })
    except Exception as e:
        return jsonify({"error": str(e)}), 400

Penjelasan Kode

  1. Inisialisasi Bobot: Bobot dan bias diinisialisasi (dalam contoh ini pre-trained, pada kasus nyata dilakukan random initialization lalu training)
  2. Fungsi Sigmoid: Mengubah nilai input menjadi 0-1, digunakan sebagai fungsi aktivasi
  3. Forward Propagation:
    • Input dikalikan dengan bobot, ditambah bias
    • Hasil dilewatkan fungsi sigmoid
    • Output hidden layer menjadi input untuk output layer
    • Proses diulang hingga menghasilkan output akhir
  4. API Endpoint: Menerima 2 input numerik, menjalankan forward propagation, mengembalikan hasil prediksi

Cara Kerja Forward Propagation

Contoh perhitungan dengan input x1=0.5, x2=0.3:

Step 1: Hidden Layer

# Hidden neuron 1
h1_input = (0.5 × 0.5) + (0.3 × 0.2) + 0.1 = 0.41
h1_output = sigmoid(0.41) = 0.601

# Hidden neuron 2
h2_input = (0.5 × -0.3) + (0.3 × 0.6) + (-0.2) = -0.17
h2_output = sigmoid(-0.17) = 0.458

# Hidden neuron 3
h3_input = (0.5 × 0.8) + (0.3 × -0.4) + 0.3 = 0.58
h3_output = sigmoid(0.58) = 0.641

Step 2: Output Layer

# Output neuron
output_input = (0.601 × 0.7) + (0.458 × -0.5) + (0.641 × 0.9) + 0.2 = 0.968
output = sigmoid(0.968) = 0.725

Hasil: Neural network memprediksi output 0.725 untuk input [0.5, 0.3]

Keunggulan & Keterbatasan

Keunggulan:

  • Dapat mempelajari pola non-linear yang kompleks
  • Adaptif terhadap data baru melalui training
  • Robust terhadap noise dalam data
  • Dapat digunakan untuk berbagai jenis masalah

Keterbatasan:

  • Membutuhkan data training yang besar
  • Proses training bisa memakan waktu lama
  • "Black box" - sulit diinterpretasi
  • Rentan terhadap overfitting
